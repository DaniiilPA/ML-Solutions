# Проекты по анализу данных и машинному обучению

В этом репозитории собраны финальные решения двух задач с хакатона (ru-code: ИИ Чемпионат).

### Состав проекта:
1. **mainA.ipynb** — Рекомендательная система (Ranking).
   * **Стек:** LightGBM, Pandas, Scikit-learn.
   * **Что сделано:** Реализован пайплайн обучения модели ранжирования контента на основе истории просмотров с применением кросс-валидации (KFold).
2. **mainB.ipynb** — Скоринг клиентов (Classification).
   * **Стек:** LightGBM, Pandas, Feature Engineering.
   * **Что сделано:** Проведен глубокий анализ признаков, обработка пропусков и обучение классификатора для предсказания отклика клиентов.

---
**Примечание:** В репозитории представлены только ноутбуки с кодом. Исходные датасеты (.csv) не загружены из-за правил конфиденциальности и большого объема данных.
